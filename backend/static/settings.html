Connected
How to get credentials ▸
  1. Go to console.cloud.google.com, create a project (or select one).
  2. Go to APIs & Services → Library, search for Photos Library API and enable it.
  3. Go to APIs & Services → OAuth consent screen.
    Under Audience, set User Type to External and publishing status to In production.
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID.
    Application type: Web application.
  5. Under Authorized redirect URIs, add exactly:
    http://localhost:8000/api/settings/google-photos/callback
  6. Copy the Client ID and Client Secret into the fields below, click Save Settings, then click Connect with Google.
Ollama (free, local AI)

Run AI summarization locally — no API key needed. Used as fallback when no Anthropic key is set.

Setup instructions ▸

Linux / Raspberry Pi: run scripts/setup-ollama-rpi.sh

Windows: run scripts\setup-ollama-windows.ps1 in PowerShell as Administrator

Manual: install from ollama.com, then ollama pull llama3.2

If Ollama runs on a different machine, replace localhost with that machine's IP.
