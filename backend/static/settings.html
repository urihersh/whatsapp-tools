    Google Photos

    Automatically upload every matched photo to Google Photos.

    Step 1 — Google Cloud credentials

    How to get credentials ▸
    1. Go to console.cloud.google.com and create a project.
    2. Enable the Photos Library API.
    3. Go to Credentials → Create Credentials → OAuth Client ID.
    4. Application type: Web application.
    5. Add authorized redirect URI: http://localhost:8000/api/settings/google-photos/callback
    6. Copy Client ID and Client Secret below, then click Save Settings.
